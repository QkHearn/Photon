# Skill: Photon Config Editor
Modify configuration files to change editor or agent preferences. 
- macOS: `~/Library/Application Support/Photon/settings.json` 
- Windows: `%APPDATA%\Photon\settings.json` 
- Linux: `~/.config/Photon/settings.json`
