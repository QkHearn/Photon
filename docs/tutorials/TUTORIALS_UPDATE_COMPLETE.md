## 🔑 关键特性

### 1. 符号搜索（AST 分析）

- **类型**: Agent 内部能力
- **触发**: 自动拦截文件读取
- **功能**: 提取代码结构（函数、类等）
- **优势**: 避免读取整个文件，节省 Token

**工作流程**:
```
LLM 请求读文件
    ↓
Agent 拦截
    ↓
AST 分析提取符号
    ↓
注入符号摘要
    ↓
LLM 选择具体符号
    ↓
精确读取代码块
```

### 2. 语义搜索

- **类型**: Agent 内部能力
- **触发**: 检测到模糊查询（空格/中文/疑问词）
- **功能**: 基于语义相似度搜索代码
- **优势**: 从模糊查询到精确定位

**工作流程**:
```
用户查询: "找到登录相关代码"
    ↓
LLM 生成可能不准确的调用
    ↓
Agent 检测模糊查询
    ↓
语义搜索 top-K 相关片段
    ↓
注入搜索结果（文件路径+行号）
    ↓
LLM 基于真实路径精确读取
```
